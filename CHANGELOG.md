# Changelog

## [2.0.0] — 2026-04-26

### Добавлено
- **3 тарифа** активной и реактивной энергии (T1/T2/T3 + сумма)
- **MQTT** — все показания + MQTT Discovery для Home Assistant, команды через топик `/cmd`
- **Web-сервер** — встроенный UI на порту 80 с Basic Auth
- **OLED SSD1306** — 4 страницы с автопролистыванием (энергия / U+I / мощность / инфо)
- **Команда 0x07** — данные абонента (лицевой счёт, ФИО)
- **Команда 0x0A** — заводские данные (серийный №, наименование прибора)
- **Команда 0x1E** — ресурс батарейки (в % остатка)
- **Кнопка переключения страниц** дисплея из Home Assistant
- **Бинарные сенсоры:** CC1101 connectivity, пломба OK, реле включено
- **HA Services:** `poll`, `relay_on`, `relay_off`
- Автоопределение числа тарифов из конфигурационного байта счётчика

### Изменено
- Архитектура компонента: плоские массивы сенсоров вместо конструктора с 40+ аргументами
- Улучшена обработка CRC (поддержка альтернативного диапазона CRC)
- Форматирование времени работы: дни + чч:мм

### Исправлено
- Знак реактивной мощности (3-байтовый вариант)
- Определение состояния реле для ПО < версии 5

## [1.0.0] — 2026-04-24
